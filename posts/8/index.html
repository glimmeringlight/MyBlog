django学习笔记(1)：Django的安装与准备

开一个新坑，django 开发。

Django 介绍

  Django 是一个开放源代码的 Web 应用框架，由 Python 写成。利用 Django 和相关的 html 和 CSS 知识，我们可以轻松构建出一个简单的网站。本文主要介绍使用 Django 的一些准备工作。

安装 Django

  首先，你需要 Python 环境。这里以 Python3 环境为例。安装有两种方式：命令行安装和使用开发软件 Pycharm。
